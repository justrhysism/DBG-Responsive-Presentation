Defining our Canvas

Flawed Process

Imagine a painter selecting their canvas, finishing their work, and then handing their work off to someone to cut into pieces and try to glue it all back together on a canvas of a different shape and size.

Which then changes for each person who looks at it...

Constraining

We define our canvas in Photoshop, immediately establishing constraints:

  • Fixing layout width, assuming minimum screen resolution
  • Defining type in pixels
  • Fixing height on certain elements

Eventually, the design is approved. We then immediately cut it to pieces and try to shove the design into our real canvas.

The Browser: Our Real Canvas

Fluid Design

Fluid Design Also known as “Liquid” design.

Width of the design is the width of the screen no matter the width.

As the width of the screen reduces, the content “flows” to fit.

These designs are based on percentages to determine widths.

Fluid design is great, but...

  • Very wide screens it looks silly
  • Very skinny screen (e.g. mobile screens) it's unreadable

What's the solution for a fluid website which is device agnostic?

Media Queries

Responsive design is a concept, yes. But it uses some rather old CSS and some very basic fundamentals to work: CSS Media Queries.

CSS Media Queries are just simple override instructions when the viewport meets certain requirements. 

body {
    font-size: 1em;
}

/* When the viewport is <979px... */
@media (max-width: 979px) {
    body {
        /* Reduce font size */
        font-size: 0.75em;
    }
}

Media Queries Simplified

We simply tell the browser what to do when the screen gets smaller (or bigger). 

body {
    /* Normally, do this */
}

/* Although, if we match this rule... */
@media (rule-to-match) {
   body {
        /* Do this */
   }
}

Can have as many Media Queries as is necessary. This particular presentation has 14 seperate Media Queries for different widths and heights.

Responsive Methods

There are various methods we can use to make a website responsive:

  • Removing

Folding

Folding is a method of folding blocks of content together.

This is the most common method for creating a responsive layout.
